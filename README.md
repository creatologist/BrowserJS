### 
```javascript
// Browser properties
+ name //String
+ version //Number

+ mobile //Boolean
+ touch	//Boolean
	
+ device //undefined -or- Object
+ device.name //String
+ device.osVersion //Number
+ device.osVersionFull //String

+ ielt9 (ie version<9) //Boolean
```
